Predicting Student Success from Early Engagement Behaviors

Project Type: Educational Data Analytics | Predictive Modeling

Analyzed 22,437 student records from the Open University Learning Analytics Dataset to investigate whether early engagement behaviors can predict course success.

Developed logistic regression models using early clicks and active learning days, created predictive probability visualizations, and identified key indicators associated with academic performance.

Dataset: Open University Learning Analytics Dataset, 22,437 student records.

Methods: Data cleaning, feature engineering, exploratory data analysis, logistic regression, odds ratio analysis, and predictive modeling.

Tools: R, ggplot2, statistical learning, and data visualization.

Skills Demonstrated: Predictive analytics, educational data mining, statistical modeling, research design, data visualization, and model interpretation.
