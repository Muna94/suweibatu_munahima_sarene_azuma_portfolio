Student Success Prediction Using Learning Analytics

Project Type: Educational Data Analytics | Predictive Modeling

Analyzed 22,437 student records from the Open University Learning Analytics Dataset (OULAD) to investigate whether early engagement behaviors can predict course success.

Developed logistic regression models using early clicks and active learning days, created predictive probability visualizations, and identified key indicators associated with academic performance.

Methods: Data cleaning, feature engineering, exploratory data analysis, logistic regression, odds ratio analysis, predictive modeling.

Dataset:Dataset: Open University Learning Analytics Dataset (OULAD), 22,437 student records.

Tools: R, ggplot2, statistical learning, data visualization.

Skills Demonstrated: Predictive analytics, educational data mining, statistical modeling, research design, data visualization, model interpretation.
